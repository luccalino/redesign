Machine-based experimenting / Programming with PsychoPy

University of Fribourg, March 2020

The scope of this compulsory seminar for Psychology students (6 ECTS) is to provide the participants with knowledge in data generation and anylsis. The workshops are interactive and are structured on 3 main points:

  • Data generation using web-based surveys (UniPark)
  • Data generation using machine-based experiments (PsychoPy in Python)
  • Data analysis (R)

Language of instruction is German.
