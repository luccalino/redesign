Visualizing sports activity data

The idea behind this project is to visualize geospatial sports activity data.

Inspired by Marcus Volz who applies mathematical data models and transforms his research into artistic visualizations, I have developed a code that scrapes data from sports activities logged on Strava by using their API.

The data that is generated allows me to render spatial visualisations based on individual cycling & running routes. Through this information, I am able to create personalized artworks of sports activities which can be shown in various combinations. Please see some examples below.

I have sourced a fabric which is made from 100% recycled PET bottles. It is of great quality, waterproof and feels like silk. I use the printed product as shower curtain.
